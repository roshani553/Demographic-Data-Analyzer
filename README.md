#  Demographic Data Analyzer

This project performs detailed **data analysis** on the [Adult Income Dataset](https://archive.ics.uci.edu/ml/datasets/adult).  
It explores demographic factors that influence income levels using **Python** and **Pandas**.

---

## Dataset Information
- Source: [UCI Machine Learning Repository - Adult Dataset](https://archive.ics.uci.edu/ml/datasets/adult)
- The dataset contains demographic details such as:
  - Age  
  - Education level  
  - Occupation  
  - Race  
  - Gender  
  - Working hours  
  - Native country  
  - Income category (â‰¤50K or >50K)

---

##  Features of the Script
- Loads and cleans the dataset  
- Displays summary statistics  
- Calculates:
  - Average age of men  
  - Percentage of people with a Bachelor's degree  
  - Income comparison between higher and lower education groups  
  - Percentage of rich people working minimum hours  
  - Country with highest percentage of rich people  
  - Most popular occupation for high earners in India  

---

## Technologies Used
- Python 3
- Pandas
